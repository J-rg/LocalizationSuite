addFiles:

Add the given files to the database. 

- (void)addFiles:(NSArray *)filenames;
Discussion

This will spawn a new action to the process manager: first all reference language files will be imported, adding files and bundles as neccessary. Second all other languages will be imported, adding localization values only.

bundleObjectWithPath:

Returns a bundle object with the given path, adding it if none with that path exists. 

- (BLBundleObject *)bundleObjectWithPath:(NSString *)path;
Discussion

The given path must be a full path. Forwards to -bundleObjectWithPath:create:includeHistory: with create YES and history NO.

bundleObjectWithPath:create:includeHistory:

Looks for a file object, creating it if wished. 

- (BLBundleObject *)bundleObjectWithPath:(NSString *)path 
        create:(BOOL)create includeHistory:(BOOL)history;
Parameters
path
A full path to the bundle.
create
If YES, the bundle will be created if it does not exists.
history
If YES, also moved bundles will be found.

createBundleObjectWithPath:

Creates a new bundle object according to the document preferences. 

- (BLBundleObject *)createBundleObjectWithPath:(NSString *)path;
Discussion

The returned object is only created and set up, no adding to internal structures are performed.

createFileObjectWithPath:

Creates a new file object according to the document preferences. 

- (BLFileObject *)createFileObjectWithPath:(NSString *)path;
Discussion

The returned object is only created and set up, no adding to internal structures are performed.

reimportLanguages:

Runs a import over all localized files in the given languages. 

- (void)reimportLanguages:(NSArray *)languages;
Discussion

The reference language is excluded. Please note that this method will also interprete previously exported reference languages strings to other languages where they have not yet been translated. Will spawn a new action to the process manager.

rescan:

Rescan the reference language and import any changes. 

- (void)rescan:(BOOL)force;
Parameters
force
If YES, all files will be imported, regardles if they changed or not.
Discussion

This will spawn a new action to the process manager, first evaluating the amount of work to do and then importing any changed files.

synchronizeObjects:forLanguages:reinject:

Writes all changes in objects to disk. 

- (void)synchronizeObjects:(NSArray *)objects forLanguages:(NSArray *)languages 
        reinject:(BOOL)reinject;
Discussion

Only the given objects will only be affected in only the given languages. If reinject is YES, all files in scope (even the not changed) will be deleted and written out. Will spawn a new action to the process manager.
