BLLanguageTranslator

A class that transfroms language names into identifiers and vice versa.

Superclass: NSObject
Declared In: BLLanguageTranslator.h

Discussion

Internally, NSLocale is heavily used, caches are created and if no direct match is found, various things are tried to get a identifier.

allLanguageIdentifiers

Returns all available identifiers. 

+ (NSArray *)allLanguageIdentifiers;
Discussion

Basically this is the list of all available language and country codes in Mac OS X. Will contain a few hundred items.

descriptionForLanguage:

The description for a language. 

+ (NSString *)descriptionForLanguage:(NSString *)language;
Return Value

A string containing the english name of the language or nil, if no such was found.

Discussion

The argument given can be both - a language name or a identifier. There is absolutely no difference between these two arguments.

identifierForLanguage:

The identifier for a language. 

+ (NSString *)identifierForLanguage:(NSString *)language;
Return Value

A string containing an identifier or nil, if no such was found.

Discussion

The argument given can be both - a language name or a identifier. There is absolutely no difference between these two arguments.

localeForLanguage:

Returns a NSLocale for a given language string. 

+ (NSLocale *)localeForLanguage:(NSString *)language;
Discussion

The argument given can be both - a language name or a identifier. There is absolutely no difference between these two arguments.
