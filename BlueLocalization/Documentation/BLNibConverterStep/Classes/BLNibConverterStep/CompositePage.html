BLNibConverterStep

A step that upgrades nib files to a newer format. Currently (Leopard) this converts nib files to the new xib format, updating the database.

Superclass: BLProcessStep
Declared In: BLNibConverterStep.h

Discussion

Basically, this is just a frontend for BLNibFileConverter. First all BLNibFileObject's will be filtered from the list, then the setps will upgrade all languages in the document.
