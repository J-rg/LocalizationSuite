BLProcessStep

The basic atomic unit of processing in an localization document.

Superclass: NSOperation
Declared In: BLProcessStep.h

Discussion

This is an subclass of NSOperation, which is always concurrent and handles all operation stuff. Subclasses normally only need to override the -perfrom and -description methods.

Discussion

descriptionChanged

Notifies that the description changed. 

- (void)descriptionChanged;
Discussion

This is a thread-safe convenience method to notify that your description changed while you are processing. Asyncronous.

finish

Notifies that the operation finished. 

- (void)finish;
Discussion

Override if you want to make some custom notifications when the step has been finished.

manager

The process manager the operation is performed by. 

- (BLProcessManager *)manager;

perform

Perform the desired action. 

- (void)perform;
Discussion

Has to be overridden, default implementation just throws. A call to this method is always encapsulated by a separate NSAutoreleasePool, so no custom memory needs to be performed in most cases.
