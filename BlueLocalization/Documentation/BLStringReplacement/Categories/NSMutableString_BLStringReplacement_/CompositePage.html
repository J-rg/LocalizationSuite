applyReplacementDictionary:reverseDirection:

Replaces all keys in dictionary with values. 

- (void)applyReplacementDictionary:(NSDictionary *)dict 
        reverseDirection:(BOOL)reverse;
Discussion

Replaces all occurences of every key with the according value. If reverse is YES, the proedure is inverted, trating values as keys and keys as values. There are no double replacements as ranges are being precomputed and replaced after that. Keys and values shouldn't be substrings of each other as the behaviour for this is undefined.
