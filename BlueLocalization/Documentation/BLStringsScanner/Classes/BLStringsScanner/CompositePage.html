processComments:

Post-processes previously scanned comments. 

+ (void)processComments:(NSMutableDictionary *)comments;
Discussion

This will apply the standard replacements and unicode replacements to both keys and values. In addition, the values will be trimmed and formatted as appropriate. It is expected (and not checked) that the mutability is given.

scanString:toDictionary:withComments:andKeyOrder:

Scans strings and comment from an given string. 

+ (BOOL)scanString:(NSString *)string toDictionary:(NSMutableDictionary *)strings 
        withComments:(NSMutableDictionary *)comments andKeyOrder:(NSMutableArray *)keys;
Return Value

Discussion

No processing is done on the data, this has to be done separately. It is expected (and not checked) that all inputs are set correctly, especially that the mutability is given to all mutable arguments.
