Filter Settings

Note:
Please note that the Localization Dictionary does not display the main window unless a dictionary file has been opened. Either load an existing glossary file or create a blank one using the „File-New“ command.




Left Language
This dropdown-list lets you decide which of your glossary/TM languages is displayed on the left side of the table below the header area.


Right Language
This dropdown-list lets you decide which of your glossary/TM languages is displayed on the right side of the table below the header area.


Filter Settings
The Filter Settings let you configure the way your glossary/TM is displayed. Click here to learn more about normalizing.


Search Field
Use the search bar to look for specific keywords in your dictionary.


Dictionary table
The dictionary table provides an overview of all entries contained in a dictionary. If you create a new dictionary, this table will be empty. The total number of entries is displayed to the right above the table.
