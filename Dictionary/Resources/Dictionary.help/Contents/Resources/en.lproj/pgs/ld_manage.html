Localization Dictionary Help

Creating, managing and editing Dictionaries

The Localization Dictionary is a tool mainly developed to manage and edit dictionaries and translation memories. Creating a new dictionary is also possible, i.e. to merge existing glossaries or to create new glossaries from various other sources.

Creating a new dictionary
Having launched the Localization Dictionary Application, click on File->New.
The Localization Dictionary now displays an emtpy dictionary which does not contain any languages or entries.

Now add your languages by using the Dictionary > Add languages menu command.
After having added your languages, you can now work with your glossary: Either add terms manually or import other glossaries.

Creating a new entry
You can add entries to the dictionary by clicking Dictionary > Add key.

Importing dictionaries
The Localization Manager allows you to import various types of dictionaries. Supported formats include XLIFF (.xlf), TMX (.tmx) and AppleGlot (.ad). You can also import dictionaries exported from the Localizer (.lod), dictionaries embedded in localizer files (.loc) and Localization Manager files (.ldb). To import a specific dictionary file, use the File-Import... command. After having imported dictionaries, check your normalizing settings in the Filter: Settings menu.

Editing a dictionary
Editing dictionary entries can be useful to forever-eliminate typos, change preset terminology or making an entry more universal. Simply double click on an entry and type your correction. Having finished your edit, simply click on another entry or hit the return button on your keyboard.

Exporting dictionaries
In case you want to use a dictionary with other tools, share it with others, or archive it, you can export it by using the File-Export menu command. The Localization Dictionary Application allows you to export your dictionary in the native .lod format compatible with all Loc-Suite applications or export as TMX.


Note:
TMX is a widely used and standardized format for translation memories developed by the LISA organization. Check out www.lisa.org for further information and a complete specification of TMX.



Using a dictionary for Concordance Search
We recommend to open all dictionaries that might prove useful for your current project. If you want to find out, whether a specific term is contained in your dictionary, type in the term in the search field of the Localization Dictionary window. Searching in the Localization Dictionary window might be useful to search for a term which might have been translated but does not show up in the suggestions since the entry does not match to the source string.
