Adding languages to the Localization Project

Once you have created the Localization Project and added your localizable resources to the file list, you want to add the language(s), you want the application localized into, to the language list in the Localization Manager window.



Click on + below the language list to open the Add Languages dialog. You can now either scroll through the list or type the desired language into the Language Name field. Add the language by clicking OK.


Useful hint:
You can either select the general language or decide to use locales for a language. The list contains locales for languages with regional differences, such as Spanish. If you want to have your application localized into Spanish and want to consider language differences between Spain and Argentina, you should add two languages: Spanish (Spain) and Spanish (Argentina). Please note that you can only select one language at a time in the Add Language dialog.


The Language status:
The left column in the Languages list states the localization progress for all listed languages. The source language (mostly English) will be shown as complete, while all other languages added will generally show nothing. Once you update any particular language with localizations (see Importing localized languages using Read In...), the status in the language list will be updated and show a percentage value (for partially finished localizations) or complete (for finished localizations).
