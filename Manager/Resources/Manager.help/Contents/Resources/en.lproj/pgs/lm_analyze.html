Analyzing the Project scope

If you want gain an overview of the project scope (and eventually project costs for external translators) the Show Statistics... window will prove very useful to you. Click Window > Show Statistics... to open the statistics window. This window is divided into three main areas Objects, Counters and Statistics.


Objects:
The Objects area provides an overview on the number of bundles, files and keys in the Localization project.

Counters:
The Counters area shows information on the number of sentences, words and characters in the source language and the target language selected in the Languages list. When selecting a different language in the Languages list, the Counters area will show how many sentences, words and characters are translated for the particular language.

Statistics:
The Statistics area shows detailed analysis information for the selected language. When selecting a language in the Languages list and clicking on Calculate in the Statistics window, the Localization Manager will perform an analysis of the language, taking into account any dictionaries selected for translation (see Embedding Dictionaries).

The analysis contains word counts for translated keys, exact matches, guesses above 75% fuzzy value, guesses above 50% fuzzy value and non-matched words. The analysis can only be performed for one language at a time. If you want to analyse another language, select this language in the Languages list and click Calculate in the Statistics window again.

If you want to analyze the project scope for a particular language, select the language in the language list within the Localization Manager main window and click Calculate in the Statistics window.

Note:
You can only perform analysis for one language at a time. If you want to analyze several languages you will need to perform an individual analysis for each language.
