Creating an empty Project

Click File > New > Empty to create a new Localization Project and choose a location and file name in the Save as dialog.



Once you have saved your new project, the Localization Manager window will appear on the screen, and the file list in the upper half will be empty.

Click + and navigate to the folder with your resources to be localized. Select the folder (the selected folder must contain a .lproj subfolder) and click Open. The files will be scanned and imported into your Localization Manager project.

Hint:
If you prefer to have your resources imported automatically from your Xcode project, please refer to chapter Creating a project from Xcode.


You will now see the resources bundle containing all the localizable files from your original .lproj folder.

Question: What file types does the Localization Manager support?
Answer:The Localization Manager can process the following files: nib, xib, strings, plist, txt, rtf, rtfd..
