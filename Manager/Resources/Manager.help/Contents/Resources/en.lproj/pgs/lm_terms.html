Definition of Terms

Term

Definition

Localization

The adaption of written content to a certain market or geographical region. Localization involves the translation of texts, adaption to local requirements such as time and date formats, and the adaption of the user interface and graphics to the local market.

Translation

The actual transcription of text from one language into another language. The translation of application texts (strings) with the Localization Suite takes place using the Localizer.

String

This Help refers to strings as enclosed text units within the application such as sentences or user interface elements. Strings as defined herein can be contained in .strings and .xib files.

Dictionary

In this Help a dictionary is defined as a translation database. This database contains entries with source language strings and the respective target language strings. The dictionary can be used to avoid double or different translation of particular sentences or words and as reference to look up previous translations. Localization Suite Dictionaries can contain several target languages. Sometimes those Dictionaries are also called glossaries or translation memories.
